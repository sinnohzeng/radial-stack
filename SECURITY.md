# 安全策略

## 支持的版本

| 版本 | 支持状态 |
|------|----------|
| 1.0.x | ✅ 安全更新 |

## 报告漏洞

如果您发现安全漏洞，请通过 GitHub Issues 以 **私密方式** 报告（使用 "Report a vulnerability" 功能）。

请勿在公开 Issue 中披露安全漏洞的细节。

我们会在 48 小时内确认收到报告，并在 7 天内提供修复方案或缓解建议。

## 安全实践

本项目遵循以下安全实践：

- 所有用户输入的文本在嵌入 SVG 前经过 XML 转义（`escapeXml`），防止 XML 注入
- 不使用任何外部 API 或网络请求
- 不收集或传输任何用户数据
- 所有依赖均为知名开源库，定期更新
